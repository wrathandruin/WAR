# Milestone 4 — Entities & Rich Interaction

## Goal
Introduce entities into the world and expand interaction depth.

## Delivered
- Basic Entity struct
- Entity storage in GameLayer
- Tile can contain entity reference
- Inspect now returns richer info (tile + entity)
- Foundation for interaction verbs

## Notes
- Not a full ECS (yet)
- Entities are lightweight and local

## Next

- entity behaviors
